## Quick Start
```lua
local Modulate = require(path.to.Modulate)

-- Create a new loader instance
local loader = Modulate.new()

-- Load all modules with Init functions
loader:getModules()

print("All modules loaded!")
```

---

## Why Modulate?

Modulate simplifies module management in Roblox by automatically loading and initializing your modules. It handles both server-side services and client-side modules with a unified API.

**Perfect for:**
- Game frameworks requiring modular architecture
- Projects with many independent services
- Clean separation between server and client code
